# Vulkan小工具依赖列表
# 汇总了所有子项目的Python依赖包

## 公共依赖包

### GUI框架
PySide6>=6.5.0

### 图像处理库
Pillow>=9.0.0

## 特定工具依赖

### 柏林噪声工具 & 高度域灰度图转换为法向量纹理图工具
numpy>=1.21.0

### 生成2D纹理数组纹理工具 & 生成立方图纹理工具
typing-extensions>=4.0.0

### 二维普通纹理转换工具（Python < 3.4）
pathlib2>=2.3.0

## 安装说明

可以使用以下命令一次性安装所有依赖：
```bash
pip install PySide6>=6.5.0 Pillow>=9.0.0 numpy>=1.21.0 typing-extensions>=4.0.0
```

或者根据具体工具需求选择安装相应的依赖包。
